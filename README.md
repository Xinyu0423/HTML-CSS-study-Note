## 单标签和双标签
1. 需要确定开始结束使用双标签，否则使用单标签

## HTML标签
1. 标题标签
   1. <h1>放入文字</h1>
   2. 最多使用到h6标题标签
   3. 重要程度依次递减
2. 段落标签
   1. <p>放入文字</p>
   2. 段落标签独占一行
   3. 如果需要换行，使用另外一个<p></p>
